# AI4PB 开发交付方法论指导书

欢迎阅读 **AI-For-Project-Building (AI4PB)** 方法论及工具指导书。

本指南旨在帮助系统工程师、架构师和开发团队，通过融合 **模型驱动架构 (MDA)**、**SCRUM 敏捷开发** 与 **AI 辅助（Copilot）** 工具，打造从需求分析、架构设计到最终代码落地与审计闭环的全新工作模式。

## 核心理念

AI4PB（AI for Project Building）的核心在于消除“架构设计”与“工程代码”之间的割裂：通过把架构和研发任务“数据化（JSON）”，并让 AI 直接吸收这些高层架构约束，从而指引 GitHub Copilot 生成不仅能运行，且**符合架构预期的代码**。
